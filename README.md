# Machine Learning

This repository contains a collection of **Jupyter notebooks** for exploring and implementing different **Machine Learning** concepts and techniques. The notebooks cover topics such as **data preprocessing**, **exploratory data analysis (EDA)**, **model training**, **evaluation**, and **visualization** across different datasets and problem types.  

## Key Libraries Used

- **NumPy** – numerical computations  
- **Pandas** – data manipulation and analysis  
- **Matplotlib** – data visualization  
- **Seaborn** – statistical plotting  
- **Scikit-learn** – machine learning models and evaluation  
- **TensorFlow / PyTorch** – deep learning experiments (where applicable)  

The repository is intended for **learning**, **experimentation**, and building practical understanding of machine learning workflows using Python.
